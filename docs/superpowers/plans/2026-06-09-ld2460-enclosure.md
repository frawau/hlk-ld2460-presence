# LD2460 Monitor-Top Enclosure Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A parametric OpenSCAD enclosure that perches the HLK-LD2460 (+ CH343P USB-serial bridge) on a monitor's top edge, antenna facing the room at a slight tilt, printed as a snap-fit front shell + rear lid.

**Architecture:** One parametric `.scad` file with a `part` switch (`shell`/`lid`/`all`) and helper modules. Verification is render-based: a pytest module shells out to the `openscad` CLI, renders each part to STL under `--hardwarnings` (manifold + warning-free), parses the binary-STL bounding box to assert overall dimensions, and writes preview PNGs for human inspection. Tests auto-skip where `openscad` is not installed.

**Tech Stack:** OpenSCAD 2021.01 (CLI), Python `pytest` (already in the venv) + `subprocess` for the render harness, binary-STL bbox parsing via `struct`.

**Conventions:** metric, parametric — every real-world dimension is a top-of-file variable. Defaults render immediately; the user re-tunes after measuring hardware. Run `black` on the Python test file. Commit per task. The `.scad` source is committed; rendered `*.stl`/`*.png` artifacts are git-ignored.

**Geometry frame:** X = width (centred on 0), Y = depth (front face at Y=0, screen side at +Y), Z = up (foot underside at Z=0; the rear hook lip extends below into −Z).

```python
import os
import shutil
import struct
import subprocess
import tempfile

import pytest

pytestmark = pytest.mark.skipif(
    shutil.which("openscad") is None, reason="openscad CLI not installed"
)

SCAD = os.path.join(os.path.dirname(__file__), "..", "enclosure", "ld2460_case.scad")
PREVIEW_DIR = os.path.join(os.path.dirname(__file__), "..", "enclosure", "preview")


def render_stl(part, defs=None):
    """Render a part to STL. Returns (CompletedProcess, stl_path)."""
    out = tempfile.NamedTemporaryFile(suffix=".stl", delete=False).name
    cmd = ["openscad", "-o", out, "--export-format", "binstl", "-D", f'part="{part}"']
    for k, v in (defs or {}).items():
        cmd += ["-D", f"{k}={v}"]
    cmd += ["--hardwarnings", SCAD]
    proc = subprocess.run(cmd, capture_output=True, text=True)
    return proc, out


def render_png(part, name):
    """Render a preview PNG into enclosure/preview/ for human inspection."""
    os.makedirs(PREVIEW_DIR, exist_ok=True)
    out = os.path.join(PREVIEW_DIR, name)
    subprocess.run(
        ["openscad", "-o", out, "-D", f'part="{part}"', "--imgsize=900,700", SCAD],
        capture_output=True,
        text=True,
    )
    return out


def stl_bbox(path):
    """Return (dx, dy, dz) bounding-box size of a binary STL."""
    with open(path, "rb") as f:
        data = f.read()
    n = struct.unpack_from("<I", data, 80)[0]
    xs, ys, zs = [], [], []
    off = 84
    for _ in range(n):
        base = off + 12  # skip the 3-float normal
        for v in range(3):
            x, y, z = struct.unpack_from("<fff", data, base + v * 12)
            xs.append(x)
            ys.append(y)
            zs.append(z)
        off += 50
    return (max(xs) - min(xs), max(ys) - min(ys), max(zs) - min(zs))


def stl_z_min(path):
    with open(path, "rb") as f:
        data = f.read()
    n = struct.unpack_from("<I", data, 80)[0]
    zmin = float("inf")
    off = 84
    for _ in range(n):
        base = off + 12
        for v in range(3):
            _, _, z = struct.unpack_from("<fff", data, base + v * 12)
            zmin = min(zmin, z)
        off += 50
    return zmin


def test_shell_renders_and_outer_size_is_sane():
    proc, stl = render_stl("shell")
    assert proc.returncode == 0, proc.stderr
    dx, dy, dz = stl_bbox(stl)
    # Outer width ≈ board width (32) + 2*fit_clear (0.8) + 2*wall (4) ≈ 36.8
    assert 35.0 <= dx <= 39.0, f"width {dx}"
    # Tall enough to hold the 49.5 mm board plus the foot
    assert dz >= 50.0, f"height {dz}"
    render_png("all", "assembly.png")
```

```scad
// Frame: X=width (centred), Y=depth (front at 0, screen at +Y), Z=up (foot base at 0)

/* ----- Measured / to-confirm (defaults are placeholders) ----- */

ld2460_t     = 1.2;    // PCB thickness            [MEASURE]
comp_height  = 5.0;    // front parts incl. perpendicular connectors [CONFIRM]
ch343_l      = 26;     // CH343P long edge (across X)
ch343_w      = 13;     // CH343P short edge (along Y)
ch343_t      = 1.2;    // CH343P thickness         [MEASURE]
usb_w        = 9.5;    // Type-C cutout width      [MEASURE]
usb_h        = 4.0;    // Type-C cutout height     [MEASURE]

wall         = 2.0;
window_wall  = 1.2;    // thin RF face
window_clear = 1.0;    // gap beyond comp_height to inner window
back_gap     = 2.0;    // behind PCB (solder side)
fit_clear    = 0.4;    // board/lid slip fit
tilt_deg     = 12;     // antenna look-down tilt
connector_clear = 12;  // perpendicular room for the plugged harness
foot_extra_depth = 4;  // foot margin behind CH343P
hook_lip_h   = 18;     // lip drop behind screen
hook_wall    = 2.5;
hook_clear   = 1.5;    // slip over the screen edge
snap_w       = 8;
snap_t       = 1.6;
snap_hook    = 1.0;
lid_t        = 2.0;

out_w     = cav_w + 2 * wall;                  // outer width
front_gap = comp_height + window_clear;        // board front -> window inner
up_in_d   = front_gap + ld2460_t + back_gap;   // upright inner depth (Y)
up_out_d  = window_wall + up_in_d;             // upright outer depth (back open)
board_margin = 3;
up_h      = ld2460_h + 2 * board_margin;       // upright inner height (board + slack)

foot_depth = up_out_d + ch343_w + foot_extra_depth + wall;

else if (part == "lid") lid();
```

```scad
// Local helper: the LD2460 board volume in the upright's local frame.
// Origin of the upright local frame is its front-bottom-left corner.
module ld2460_board_cavity() {
    // Hollow interior of the upright (board + clearances), leaving the thin
    // window at the front and walls on the sides; back stays open for the lid.
    translate([wall, window_wall, wall])
        cube([out_w - 2 * wall, up_in_d + 1, up_h - 2 * wall]);
}

module ld2460_edge_slots() {
    // Two grooves that grip the left/right PCB edges, positioned behind the
    // component+window gap so the board's front (antenna) face sees the window.
    slot_y = window_wall + front_gap;          // front face of PCB
    slot_d = ld2460_t + fit_clear;             // groove depth in Y
    slot_z0 = wall + board_margin;             // board bottom
    for (sx = [wall - 0.6, cav_w + 2 * wall - wall - 0.6])
        translate([sx, slot_y, slot_z0])
            cube([0.6, slot_d, ld2460_h]);     // thin rib pair forms the slot walls
}

module foot_interior() {
    translate([wall, wall, wall])
        cube([out_w - 2 * wall, foot_depth - 2 * wall, foot_h]);  // open top
}

module shell() {
    difference() {
        // Solid blank
        union() {
            translate([-out_w / 2, 0, 0]) cube([out_w, foot_depth, foot_h]);
            translate([0, 0, foot_h])
                rotate([tilt_deg, 0, 0])
                    translate([-out_w / 2, 0, 0]) cube([out_w, up_out_d, up_h]);
        }
        // Foot interior
        translate([-out_w / 2, 0, 0]) foot_interior();
        // Upright cavity (board space), in the tilted local frame
        translate([0, 0, foot_h])
            rotate([tilt_deg, 0, 0])
                translate([-out_w / 2, 0, 0]) ld2460_board_cavity();
    }
    // Edge slots (additive ribs) inside the upright
    translate([0, 0, foot_h])
        rotate([tilt_deg, 0, 0])
            translate([-out_w / 2, 0, 0]) ld2460_edge_slots();
}
```

````markdown
# LD2460 Monitor-Top Enclosure

Parametric OpenSCAD case that perches the HLK-LD2460 (+ CH343P USB-serial bridge)
on a monitor's top edge, antenna facing the room at a slight tilt. Prints as two
snap-fit parts: a front **shell** and a rear **lid**.

## Render / export

```bash
# Preview in the GUI
openscad enclosure/ld2460_case.scad

# Export each printable part to STL
openscad -D 'part="shell"' -o shell.stl enclosure/ld2460_case.scad
openscad -D 'part="lid"'   -o lid.stl   enclosure/ld2460_case.scad
```

`part` selects `"shell"`, `"lid"`, or `"all"` (assembled preview). Override any
parameter on the command line with `-D name=value`.

## Measure these before printing

The defaults render, but confirm against your hardware (edit the top of
`ld2460_case.scad`):

- `ld2460_t` — LD2460 PCB thickness.
- `comp_height` — tallest front-side stack **including the perpendicular header
  pins** (≈ 5 mm; the wires plug in perpendicular, hence `connector_clear`).
- `ch343_t`, `usb_w`, `usb_h` — CH343P PCB thickness and Type-C connector size.
- `screen_edge_t` — your monitor's top-edge thickness (sets the hook gap).
- `tilt_deg` — antenna look-down angle (default 12°).

## Print settings

- Material: **PETG / ASA / ABS** (avoid PLA near a warm/sunny screen).
- Shell printed **window-face-down** (smooth RF face); lid printed flat.
- 0.2 mm layers, 3 perimeters, 20–30% infill.
- Keep the front (window) face free of metal/metallic paint — 24 GHz passes
  through thin plastic but not metal.

See `../docs/hardware-enclosure-notes.md` for wiring, pinout, and board data.
````
